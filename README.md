## How to Use
1. Fork Repo
2. Change `RATE` (1-10000) in `./src/WETHMarket.sol` (denominated in BPS e.g. `9500` means 1 WETH for 0.95 ETH)
3. Deploy
4. Allow people to sell WETH for ETH
